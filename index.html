FREQUENTLY ASKED QUESTIONS

  • arrow_rightWhat is the price per pound for airfreight?

    -All air packages with a minimum weight of 1lb will have a unit cost of $2.50 per pound.


    -The minimum weight is 1lb and the rate is calculated based on 1lb units. For example if the package weighs 2.3lbs 3lb is posted.


    -If there is a 10lbs difference between the actual weight and the volumetric weight it is charged at volumetric weight rate.

  • arrow_rightHow is the volumetric weight calculated?

    -The volumetric weight is calculated using the following package measurements in inches:

    (Length x Width x Height) / 166


    -The volumetric weight fee is $2.00 per pound and applies only if there is a 10lbs difference from the actual weight.
